# ASM-Tetris
Full Tetris game created in ASM emulated in a raspberry pi 3b with QEMU. This project was created with 2 of my CS classmates.

Run with $make runSGPIOM in one terminal and $make runQEMU in a different one.

Rotate     - W
Move Right - D
Move Left  - A
Hard Drop  - S
Restart    - SPACE
